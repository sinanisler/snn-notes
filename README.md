# SNN Notes

A modern, feature-rich note-keeping system for WordPress admin with tags, folders, templates, search, export, revisions, and collaboration features.

## Features

- **Modern Editor**: Built with Quill.js for a rich editing experience.
- **Organization**: Use tags and folders to keep your notes organized.
- **Fast & Reactive**: AJAX-based interface for smooth interaction.
- **Auto-Save**: Never lose your work with automatic saving.
- **REST API**: Full REST API support for external integrations.

## Installation

1. Upload the `snn-notes` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage

1. Navigate to the **SNN Notes** menu in your WordPress admin dashboard.
2. Click **New Note** to create a new note.
3. Use the sidebar to manage tags and navigate your notes.
